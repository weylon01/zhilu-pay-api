# 指路支付后端部署指南

## 一、项目概述

本项目是指路教育咨询 APP 的微信支付后端 API，提供以下功能：
- ✅ 创建支付订单（统一下单）
- ✅ 处理微信支付回调
- ✅ 查询订单状态
- ✅ 订单管理

**技术栈：**
- ASP.NET Core 8.0
- SQLite 数据库
- Senparc.Weixin SDK
- RESTful API

## 二、本地开发环境搭建

### 2.1 安装 .NET 8.0 SDK

1. 下载并安装 .NET 8.0 SDK
   - 访问：https://dotnet.microsoft.com/download
   - 选择 Windows 版本安装

2. 验证安装
   ```powershell
   dotnet --version
   ```
   应该显示 8.0.x

### 2.2 配置项目

1. **修改配置文件**

   打开 `appsettings.json`，填写您的微信商户信息：
   
   ```json
   {
     "WeChatPay": {
       "AppId": "wxb73987ed222a5232",
       "MchId": "1724917960",
       "ApiKey": "您的32位API密钥",
       "NotifyUrl": "http://localhost:5000/api/payment/notify"
     }
   }
   ```

2. **获取 API 密钥**
   - 登录微信商户平台：https://pay.weixin.qq.com
   - 进入【账户中心】>【API 安全】>【API 密钥】
   - 复制您的 32 位密钥

### 2.3 本地运行

```powershell
# 进入项目目录
cd d:\千问\ZhiluPayApi

# 恢复依赖包
dotnet restore

# 运行项目
dotnet run
```

访问：http://localhost:5000/swagger 查看 API 文档

## 三、服务器部署方案

### 3.1 选择云服务器

推荐以下服务商：

| 服务商 | 推荐配置 | 月费用 |
|--------|---------|--------|
| 阿里云 | 2 核 4G | ~100 元 |
| 腾讯云 | 2 核 4G | ~80 元 |
| 华为云 | 2 核 4G | ~90 元 |

**最低配置要求：**
- CPU：1 核
- 内存：2GB
- 系统盘：40GB
- 带宽：1Mbps

### 3.2 购买服务器（以腾讯云为例）

1. **访问腾讯云控制台**
   - https://console.cloud.tencent.com

2. **选择轻量应用服务器**
   - 地域：选择靠近用户的地区（如广州）
   - 镜像：Ubuntu 22.04 LTS
   - 套餐：2 核 4G 60GB

3. **配置网络**
   - 开放端口：80（HTTP）、443（HTTPS）、5000（API）

4. **设置密码**
   - 记录服务器 IP 地址
   - 记录 root 密码

### 3.3 服务器环境配置

#### 步骤 1：连接服务器

```bash
# Windows 用户可以使用 PowerShell 或 PuTTY
ssh root@你的服务器IP
```

#### 步骤 2：安装必要软件

```bash
# 更新系统
apt update && apt upgrade -y

# 安装 .NET 8.0
wget https://dot.net/v1/dotnet-install.sh
chmod +x dotnet-install.sh
./dotnet-install.sh --channel 8.0

# 配置环境变量
echo 'export DOTNET_ROOT=$HOME/.dotnet' >> ~/.bashrc
echo 'export PATH=$PATH:$DOTNET_ROOT:$DOTNET_ROOT/tools' >> ~/.bashrc
source ~/.bashrc

# 验证安装
dotnet --version

# 安装 Nginx（反向代理）
apt install nginx -y

# 安装 SQLite
apt install sqlite3 -y
```

#### 步骤 3：部署应用

```bash
# 创建部署目录
mkdir -p /opt/zhilu-pay
cd /opt/zhilu-pay

# 从本地上传代码（在本地电脑执行）
# scp -r d:\千问\ZhiluPayApi root@服务器IP:/opt/zhilu-pay

# 或者使用 Git
# git clone https://your-git-repo/ZhiluPayApi.git

# 发布项目
dotnet publish -c Release -o ./publish

# 创建 systemd 服务
cat > /etc/systemd/system/zhilu-pay.service << EOF
[Unit]
Description=Zhilu Pay API Service
After=network.target

[Service]
Type=notify
User=root
WorkingDirectory=/opt/zhilu-pay/publish
ExecStart=/root/.dotnet/dotnet Zhilu.PayApi.dll
Restart=always
RestartSec=10
KillSignal=SIGINT
SyslogIdentifier=zhilu-pay
Environment=ASPNETCORE_ENVIRONMENT=Production
Environment=DOTNET_PRINT_TELEMETRY_MESSAGE=false

[Install]
WantedBy=multi-user.target
EOF

# 启动服务
systemctl daemon-reload
systemctl enable zhilu-pay
systemctl start zhilu-pay

# 查看状态
systemctl status zhilu-pay
systemctl logs -f zhilu-pay
```

#### 步骤 4：配置 Nginx 反向代理

```bash
# 创建 Nginx 配置
cat > /etc/nginx/sites-available/zhilu-pay << 'EOF'
server {
    listen 80;
    server_name yourdomain.com;

    location / {
        proxy_pass http://localhost:5000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection keep-alive;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
    }
}
EOF

# 启用配置
ln -s /etc/nginx/sites-available/zhilu-pay /etc/nginx/sites-enabled/
rm -f /etc/nginx/sites-enabled/default

# 测试配置
nginx -t

# 重启 Nginx
systemctl restart nginx
```

#### 步骤 5：配置 HTTPS（推荐）

```bash
# 安装 Certbot
apt install certbot python3-certbot-nginx -y

# 申请证书
certbot --nginx -d yourdomain.com

# 自动续期
certbot renew --dry-run
```

### 3.4 配置防火墙

```bash
# 如果使用腾讯云/阿里云，需要在控制台开放端口
# 如果使用 UFW：
ufw allow 80/tcp
ufw allow 443/tcp
ufw allow 22/tcp
ufw enable
```

## 四、域名配置

### 4.1 购买域名

推荐服务商：
- 阿里云：https://wanwang.aliyun.com
- 腾讯云：https://dnspod.cloud.tencent.com

### 4.2 域名备案

**重要：** 微信支付回调地址必须是已备案的域名！

备案流程：
1. 登录云服务商控制台
2. 找到"ICP 备案"
3. 按照指引提交资料
4. 等待审核（通常 1-2 周）

### 4.3 DNS 解析

在域名管理控制台添加 A 记录：
```
类型：A
主机记录：@
记录值：你的服务器 IP
TTL：10 分钟
```

## 五、配置微信支付回调

### 5.1 修改配置

编辑 `/opt/zhilu-pay/publish/appsettings.json`：

```json
{
  "WeChatPay": {
    "AppId": "wxb73987ed222a5232",
    "MchId": "1724917960",
    "ApiKey": "您的实际 API 密钥",
    "NotifyUrl": "https://yourdomain.com/api/payment/notify"
  }
}
```

### 5.2 重启服务

```bash
systemctl restart zhilu-pay
```

### 5.3 在微信商户平台配置

1. 登录微信商户平台
2. 进入【产品中心】>【开发配置】
3. 配置支付回调地址：`https://yourdomain.com/api/payment/notify`

## 八、常见问题

### Q1: 服务无法启动
```bash
# 查看详细日志
journalctl -u zhilu-pay -n 100

# 检查端口是否被占用
netstat -tlnp | grep 5000
```

### Q2: 微信支付回调失败
1. 检查域名是否已备案
2. 检查服务器能否外网访问
3. 查看日志确认回调是否到达

### Q3: 数据库文件不存在
```bash
# 手动创建
touch /opt/zhilu-pay/publish/zhilu.db
chmod 666 /opt/zhilu-pay/publish/zhilu.db
systemctl restart zhilu-pay
```
